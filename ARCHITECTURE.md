# Burn Budgeter Architecture

Our application is built on AWS.
We use a **t3.micro** instance for the API server.
Database is **RDS PostgreSQL** (db.t3.micro).
Storage is in **S3**.

For AI, we use **GPT-4o** for advanced logic and **Gemini 1.5 Pro** for light tasks.
We also use **Claude 3.5 Sonnet**.
